> 全成本拆解系列·第3篇。前两篇讲了地价和拆迁费，这篇讲开发成本里最容易「签着签着就漏一块」的科目，前期工程费。

前期工程费有个别的科目没有的特点，它下面的合同，基本都是按建筑面积单方签的。设计费多少钱一平米，勘察多少钱一平米，全过程咨询多少钱一平米。所以这个科目特别适合拿单方说话，你手里有几个真实项目的单方，跟设计院谈判、做拿地测算，心里就有底。

这个项目前期工程费，合约规划2,620万、动态成本定格2,711万，折单方111到115元/㎡，分母是总建面23.66万平米。另一个温州住宅项目的土增税清算终稿里，这个科目落在273元/㎡，口径是可售面积。差出一倍多，不是谁错了，是七通一平那一块的场地条件差异，后面讲。

土增税清算口径下，前期工程费是开发成本第2小项，法定定义在《土地增值税暂行条例实施细则》第七条，规划、设计、项目可行性研究、水文地质勘察、测绘、三通一平等支出。落到项目上就是五摊钱，规划设计费、勘察测绘费、报建规费、临时设施费、三通一平。

先看这个项目设计费的真实构成。一套完整的设计，要按专项拆开签六七份合同，

| 设计专项 | 金额(万元) | 折建面单方(元/㎡) |
|---|---:|---:|
| 方案设计 | 约355 | 15.0 |
| 施工图设计 | 约467 | 19.7 |
| PC(装配式)深化设计 | 约78 | 3.3 |
| 景观设计 | 约88 | 3.7 |
| 幕墙设计 | 约10 | 0.4 |
| 精装设计 | 约35 | 1.5 |
| 上述专项小计 | 约1,033 | 43.7 |
| 实际已签设计类合计(含零星分项) | **约1,187** | **50.2** |

设计之外已签的，勘察测绘约86万(3.6元/㎡)，全过程工程咨询约214万(9.0元/㎡)，合计约1,487万。对照合约规划总盘2,620万，差额约1,133万是报建规费、临时设施、三通一平这些，随开工陆续签，不是漏项，是节点未到。

## 各子项的常规单方区间

把两个温州真实项目(一个合约规划口径、一个清算终稿口径)摆在一起，再加上行业惯例，浙江住宅项目按建面单方签约的常见区间大致是这样，

| 子项               | 常见区间(元/㎡建面) | 真实锚点                      | 备注                            |
| ---------------- | ----------- | ------------------------- | ----------------------------- |
| 方案+施工图设计(建筑结构机电) | 30~60       | 本项目35                     | 房企集采压价后的行情，市场化单体项目会更高         |
| 景观设计             | 3~8         | 本项目3.7                    | 按景观面积签的要换算                    |
| 精装设计             | 每精装㎡20~40   | 本项目折总建面1.5                | 区间按精装面积计价，锚点是分摊到总建面后的数，两个口径别混 |
| PC/幕墙/人防等专项深化    | 每项1~5       | PC 3.3、幕墙0.4              | 专项越多笔数越碎                      |
| 设计类合计            | 40~80       | 本项目50、另一项目68(清算口径)        |                               |
| 勘察(详勘+补勘)        | 3~12        | 另一项目10.8                  | 地质复杂取上限                       |
| 测绘(放线/预测绘/竣工实测)  | 2~8         | 另一项目6.1                   | 贯穿全周期好几笔                      |
| 可研+规划报批类         | 0.5~2       | 另一项目约0.5                  |                               |
| 检测(桩基/主体/见证取样)   | 5~15        |                           | 常挂在建安或工程建设其他费下，口径先说清          |
| 报建规费(行政性收费)      | 20~60       |                           | 一地一策，人防易地建设费另算                |
| 临时设施+三通一平        | 30~190      | 另一项目七通一平187               | 场地条件决定，弹性最大                   |
| **前期费用总盘**       | **100~280** | 本项目115、另一项目273、同城商办项目取200 |                               |

这张表里最值得盯的是最后一行的跨度。同样叫前期工程费，115和273能差一倍多，差就差在七通一平。场地平整、换填、临时道路这些钱跟着地块的原始条件走，一块平整的净地和一块要大面积换填塘渣的软基地，前期费用不是一个量级。拿别人项目的前期单方做测算，先问一句场地什么条件。

检测费多说一句。桩基检测、主体结构检测、材料见证取样，这些钱法定归属没有明列，实操里有的挂建安、有的挂工程建设其他费，还有的进前期。金额不大，但测算和清算时要跟着自己项目的归集口径走，别两头都计或者两头都漏。

## 这个科目的三个坑

设计费按专项分开签，最容易漏签。方案、施工图、幕墙、精装、景观、PC深化，六七份合同，甲方只盯着方案加施工图两个大头的话，幕墙设计、PC深化这种专项到了要出图的时候才发现没签，变成边设计边补合同，谈判筹码全丢了。

报建规费种类杂，拿清单逐项核对最靠谱。规划、人防、消防、白蚁防治、防雷检测，每个城市每个环节的收费项目和标准都不一样，测算阶段拿经验估个总数糊弄过去，实际报建时冒出来两三项没算的，单项金额不大，加起来几十上百万。

全过程工程咨询费容易被当成软性支出砍掉。这个项目全咨214万，花在前期，买的是设计、成本、进度的统筹，真砍掉往往在后期以变更、返工的形式加倍花回来。这里还有个归口细节，同样叫咨询费，规划咨询、绿建咨询、节能评估这类技术咨询跟着服务对象进前期工程费，而工程造价咨询费在真实清算案例里进的是开发间接费下的其他间接费用，跟监理费一个口袋。归哪个扣除项，认税务口径，不认合同名字。

税务口径收个尾。前期工程费属于开发成本，与地价合计后加计扣除20%。清算时税务机关逐笔核对合同和发票，按专项分开签的设计费，合同和结算资料也要按专项分开留存，别图省事混记在一个总账科目下，到时候说不清哪笔钱对应哪份图纸。
